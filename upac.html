UPAC

            UPAC tarafımca python dilinde GNU/Linux için yazılmış tarafımca 
            yazmış olduğum, belli sıkıştırma
            algoritmaları ile dosyaları ".upac" formatında
            paketleyip,bunları kurma işlevine sahip
            bir programdır. Dosyaları yüksek sıkıştırma oranları
            ile sıkıştıran bir programdır. Güncellemeleri Youtube
            kanalımda yayınlamaktayım. Aşağıdaki panelden
            ulaşabilirsiniz.
            Örnek kodlar şunlardır:

YardIm için:

upac -h

Versiyon için:

upac version

Güncelleme için:

sudo upac update

Paket indirmek için:

sudo upac install {paket_adı}

Paket üretmek için:

sudo upac make

UPAC'I indirmek için tIkla!
